💉 IUI Process ఎలా ఉంటుంది?

మీ ovulation track చేస్తాం (ultrasound + blood test). Ovulation time కి husband sperm collect చేసి, lab లో wash చేస్తాం. Fine catheter తో directly uterus లోకి inject చేస్తాం. 15 నిమిషాలు మాత్రమే — pain లేదు, anaesthesia అవసరం లేదు.

🎯 IUI ఎవరికి Suitable?

Tubes open అయినప్పుడు. Husband sperm count mild గా తక్కువగా ఉన్నప్పుడు. 1–2 సంవత్సరాలు try చేసినా pregnancy రాకపోయినప్పుడు (unexplained). PCOS తో ovulation stimulate చేసిన తర్వాత. Cervical mucus problem ఉన్నప్పుడు.

📊 Success Rate ఎంత?

Natural cycle IUI: 10–15% per attempt. Medicated IUI (stimulation తో): 20–30%. 3 cycles try చేస్తాం — work అవ్వకపోతే IVF కి move అవుతాం. Age తక్కువగా ఉంటే success better గా ఉంటుంది.

🔄 ఎన్ని IUI Cycles Try చేయాలి?

సాధారణంగా 3 cycles try చేస్తాం. 3 లో work అవ్వకపోతే IVF better option అవుతుంది. Dr. Prashanthi గారు ప్రతి cycle తర్వాత review చేస్తారు — next step clearly చెప్తారు.

💊 Medicated IUI అంటే ఏంటి?

IUI తో పాటు ovaries stimulate చేయడానికి tablets (letrozole / clomiphene) ఇస్తాం. ఇది 1–2 follicles develop అవ్వడానికి help చేస్తుంది. Success rate natural IUI కంటే better ఉంటుంది.

తరచుగా అడిగే Questions

IUI painful గా ఉంటుందా?
లేదు — IUI చాలా mild procedure. Catheter uterus లోకి వెళ్ళేటప్పుడు కొంచెం cramp feel అవ్వవచ్చు — period cramps లాంటిది. Anaesthesia అవసరం లేదు. Procedure అయిన తర్వాత 15–20 నిమిషాలు rest తీసుకుని home కి వెళ్ళవచ్చు.
IUI ki ఎంత cost అవుతుంది Hyderabad లో?
Medicated IUI (tablets + monitoring scans + IUI procedure) అన్నీ కలిపి మదర్ హాస్పిటల్స్ లో affordable price లో ఉంటుంది. Exact amount కోసం 97059 93366 call చేయండి — hidden charges లేవు.
IUI తర్వాత ఎప్పుడు pregnancy test చేయాలి?
IUI తర్వాత 14 రోజులకు blood pregnancy test (beta-hCG) చేయించుకోండి. Urine home test accurate కాదు — blood test accurate గా ఉంటుంది. 14 రోజుల ముందు test చేస్తే wrong result రావచ్చు.
IUI fail అయితే directly IVF కి వెళ్ళాలా?
3 IUI cycles fail అయిన తర్వాత IVF discuss చేస్తాం. కానీ age (35+), tube block, severe male factor ఉంటే — IUI skip చేసి directly IVF better. Dr. Prashanthi గారు మీ specific situation బట్టి honest గా advise చేస్తారు.
IUI cycle ఎంత కాలం పడుతుంది?
ఒక IUI cycle మీ period నుండి 2–3 వారాలు. Day 2–3 baseline scan → 5–7 రోజులు tablets → monitoring scan → ovulation trigger → IUI procedure → 14 రోజుల తర్వాత pregnancy test. Total: approximately 3–4 వారాలు.
Husband sperm count ఎంత ఉంటే IUI try చేయవచ్చు?
Wash తర్వాత minimum 5–10 million motile sperm ఉంటే IUI suitable. Count అంతకంటే తక్కువగా ఉంటే (severe male factor) IVF+ICSI better option. Semen analysis report చూసిన తర్వాత Dr. Prashanthi గారు clarify చేస్తారు.
